## Install
```commandline
git clone https://github.com/MichaelTMatthews/CraftaxExperiments.git
cd Craftax_Baselines
pip install -r requirements.txt -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html
pre-commit install
```
